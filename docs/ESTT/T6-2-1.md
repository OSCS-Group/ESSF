# 威胁类型：违规使用软件
- 威胁编号：T6-2-1
- 威胁分类：违规操作-违规使用
- 威胁描述：组织或个人违反软件使用许可协议，包括超出授权范围使用、未经授权复制、修改或分发软件等行为。

# 威胁案例
## 闭源应用软件
- 2018年NetApp违规使用Oracle事件：NetApp被指控超出许可范围使用Oracle的软件，双方最终以2500万美元和解。


# 解决方案
1. 许可管理
   - 建立软件许可管理
   - 实施使用监控
   - 定期合规审查

2. 使用控制
   - 部署使用控制机制
   - 实施访问限制
   - 建立使用审计

3. 合规管理
   - 制定使用政策
   - 开展合规培训
   - 建立违规处理机制
