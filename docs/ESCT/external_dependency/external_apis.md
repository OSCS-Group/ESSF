# 外部 API

> 外部API是外部依赖层中的重要资产类型之一，它为软件应用提供了与第三方服务集成的能力。

## <b>定义</b>

外部API是由第三方服务提供商提供的应用程序编程接口(API)，用于集成外部功能或数据到企业自身的软件应用中。这些API允许应用程序通过标准化的方式与外部系统进行通信和数据交换，从而扩展应用的功能，而无需自行开发这些功能。
