# 自研组件

> 自研组件是业务应用层中的重要资产类型之一，它是企业内部团队开发的、可复用的软件模块或库，提高了软件开发的效率和一致性。

## <b>定义</b>

自研组件是企业内部团队开发的、可复用的软件模块或库，这些组件封装了特定的功能或业务逻辑，可以在企业内部的多个项目或应用中重复使用。自研组件通常针对企业特定需求设计，体现了企业的技术积累和最佳实践。
