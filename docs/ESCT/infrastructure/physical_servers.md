# 物理服务器

> 物理服务器是基础设施层中的核心资产类型之一，它为软件应用提供了最基础的计算和存储能力。

## <b>定义</b>

物理服务器是指企业自建或租用的物理硬件服务器，用于运行软件应用及相关服务。这些服务器通常安装在企业数据中心或托管机房中，由专业团队进行管理和维护。物理服务器是最基础的计算资源提供者，为上层的虚拟化环境、容器和应用程序提供硬件支持。
