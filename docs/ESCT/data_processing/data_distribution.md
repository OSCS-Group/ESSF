# 数据分发

> 数据分发是数据处理层中的重要资产类型之一，它为系统间的数据传输和共享提供了可靠的机制。

## <b>定义</b>

数据分发是用于在不同系统之间传输和分发数据的软件平台，常用于构建数据管道，实现数据共享和集成。这类系统通常采用消息队列或事件流的方式，支持数据的实时传输和异步处理，确保数据能够可靠地从源系统传递到目标系统。
