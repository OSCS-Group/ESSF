# 对象存储

> 对象存储是数据处理层中的重要资产类型之一，它专门用于存储和管理海量非结构化数据。

## <b>定义</b>

对象存储是用于存储海量非结构化数据的软件，擅长存储各种文件类型，如图片、视频和文档。它将数据作为对象进行存储，每个对象包含数据本身、元数据和唯一标识符。对象存储通常采用扁平的命名空间，不同于传统的文件系统层次结构，这使其特别适合于大规模数据存储和云环境。
